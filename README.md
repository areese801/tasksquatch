# tasksquatch

> An offline-first, local-only todo tracker. The tracker that hunts down your tasks.

`tasksquatch` is a single-user Python todo app inspired by Todoist, with no SaaS dependency. It exposes five surfaces over one shared core: CLI, TUI, Web UI, REST API, and an MCP server for Claude Code.

**Status:** under construction. See [`docs/spec.md`](docs/spec.md) for the design and the planned v1 surface.
